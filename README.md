# 🛡️ Aegis-Termux EDR
**Advanced Deception-Based Security Framework for Android (Termux)**

Aegis-Termux is a proactive security tool designed to trap intruders. It uses **Active Deception** by creating honey-folders (baits) to lure attackers, while simultaneously securing and encrypting real user data in the background.

## 🔥 Features
* **Decoy System:** Strategic honeypot directories to bait unauthorized access.
* **Stealth Encryption:** Instant AES-256 encryption of sensitive data (Gallery, Downloads, etc.) upon breach.
* **Intruder Profiling:** Captures intruder IP and silent camera photos.
* **Instant Alerts:** Sends full security reports directly to your Telegram Bot.
* **Anti-Kill Protection:** Hardened process that resists normal termination signals.
